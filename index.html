Havenly Property Rental Platform

Havenly is a full-stack property rental platform inspired by Airbnb that enables users to discover, list, and review unique places to stay. The application provides a complete booking-style experience with secure authentication, interactive maps, image uploads, and role-based access control.

I built Havenly to gain hands-on experience with developing production-ready web applications, focusing on backend architecture, authentication, database design, and third-party API integrations. The project implements full CRUD operations, server-side validation, session-based authentication, cloud image storage, and location-based property visualization while maintaining a responsive and user-friendly interface.

Highlights

  • Secure authentication and authorization using Passport.js and session management.
  • Property listing management with full CRUD functionality.
  • Cloudinary integration for efficient image uploads and storage.
  • Interactive property locations powered by the Mapbox API.
  • Review and rating system with ownership-based access control.

Tech Stack

Node.js • Express.js • MongoDB Atlas • Passport.js • EJS • Bootstrap • Cloudinary • Mapbox API • Joi

Challenges & Learnings

  • Integrated multiple third-party services (Cloudinary and Mapbox) into a unified application.
  • Structured the backend using the MVC architecture to keep the codebase modular and maintainable.
  • Improved understanding of RESTful API design, MongoDB relationships, and middleware.

Kiwi SQFT Property Management & Admin Dashboard

Kiwi SQFT is a production-ready full-stack real estate platform developed for a live client to simplify property discovery, agent interactions, and platform management. The application provides a seamless experience for users to browse properties while offering dedicated dashboards and administrative tools for managing the platform.

The project was built with a strong focus on scalability, maintainability, and real-world workflows. It features secure authentication, role-based access control, responsive dashboards, and a comprehensive admin panel for managing users, agents, properties, leads, and engagement data.

Highlights

  • Implemented secure authentication and RBAC using Supabase Authentication.
  • Developed dedicated dashboards for users and administrators with protected routing.
  • Created an admin panel for managing users, agents, properties, leads, and platform activity.
  • Designed a responsive single-page application with reusable React components and modern routing.

Tech Stack

React • TypeScript • Vite • Tailwind CSS • React Router • Supabase • PostgreSQL • Context API

Challenges & Learnings

  • Designed and implemented strict role-based access control to separate user and admin experiences.
  • Built scalable dashboard layouts and reusable UI components to improve maintainability.
  • Learned to structure a production-ready React application using modern routing and Context API.
03.

Vyom Health Healthcare Portal

Vyom Health is a digital health record management platform developed for the Smart India Hackathon (SIH) to improve healthcare accessibility for migrant workers in Kerala. The platform provides each worker with a unique digital health ID linked to a QR code, enabling secure, portable access to medical records and ensuring continuity of care across healthcare providers.

Beyond digital record management, the proposed solution includes AI-powered disease assistance, government health analytics dashboards, interactive disease heat maps, and predictive outbreak alerts to support data-driven public health decisions. The platform is designed to align with India's digital health initiatives through integration with the ABHA ecosystem.

Highlights

  • Designed a QR code-based digital health ID for secure and portable medical records.
  • Planned AI-powered symptom analysis for preliminary disease suggestions.
  • Designed government dashboards with real-time health insights and analytics.
  • Conceptualized predictive outbreak alerts and interactive disease heat maps.

Tech Focus

HealthTech • Digital Health Records • QR Code System • AI-Assisted Healthcare • Data Analytics • Public Health Intelligence • ABHA (NDHM)
